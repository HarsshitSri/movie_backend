# Movie Backend

A production-ready RESTful backend application for managing movies, user authentication, ratings, reviews, favorites, and watchlists.

## Tech Stack

- Java 26
- Spring Boot
- PostgreSQL
- Maven
- Docker
- Spring Security
- JWT
- Hibernate / JPA

## Features

- User Authentication & Authorization
- Role-Based Access Control
- Movie Management
- Ratings
- Reviews
- Favorites
- Watchlists
- Search & Filtering
